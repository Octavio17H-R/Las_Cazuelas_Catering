Las Cazuelas - Sitio Web de Catering

Sitio web moderno para Las Cazuelas, un servicio de catering especializado en tacos y comida mexicana para eventos.
Diseñado con enfoque visual, experiencia de usuario y conversión de clientes.

Características
Diseño moderno y responsive
Menú interactivo de tacos
Integración con redes sociales
Animaciones suaves con Framer Motion
Hero con imagen atractiva
Navegación rápida con React Router
Tecnologías utilizadas
React
Vite
Tailwind CSS
Framer Motion
Lucide Icons
Instalación

Instala dependencias:

npm install

Corre el proyecto:

npm run dev
Estructura del proyecto
src/
│── assets/        # Imágenes (logo, hero, etc.)
│── components/    # Componentes reutilizables
│── pages/         # Páginas principales
│── App.jsx        # Configuración principal
│── main.jsx       # Punto de entrada
